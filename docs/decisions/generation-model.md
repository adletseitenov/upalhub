## Рекомендация (ЧЕРНОВИК)

Гибрид: **text-секции — оставить flash** (100% onTopic/language, дешевле в 4–5×, keyCorrect-провалы сосредоточены в audio); **audio-секции (modality='audio') — генерить pro** (100% passageAnswerable, 85% keyCorrect): passage↔вопрос-согласованность — самое сложное место, а объём audio-заданий мал. Альтернатива: всё на pro (+качество ключей, ×4 цена, амортизируется банком). Qwen — не сейчас (недоступен стабильно; при появлении — прогнать `npm run eval:tasks`, отдельно смотреть казахскую метрику).
